SUMMARY OF PROJECT

 I have developed an eye-disease diagnostic website called ABHIN Eye Center, that diagnoses eye diseases based on symptoms written in normal sentences by the user. My website can perform diagnoses for more than ten different eye diseases. Diagnosis is done based on the following steps:

 The first step is to obtain the text entered by the user and convert it into a String array, which contains only words. Using regular expressions, my code removes all numeric characters, tabs, spaces, and all symbols from the String array.

 Then, each word in the String array is compared with the available list of symptoms present in the JavaScript code of the website. Three powerful functions convert the obtained symptom or symptoms into a numerical number that contains only ‘1’s and ‘0’s.

 Also, the website scans the text for the presence of any ‘negative’ symptoms. For instance, the phrase “no peripheral vision” is read differently than the phrase “peripheral vision”. The first symptom “no peripheral vision” gives a diagnosis of Glaucoma, while the second symptom “peripheral vision” gives a diagnosis of an extraneous disease; a diagnosis of an extraneous disease is given when the text given by the user is not useful enough to make any viable diagnosis.

 Finally, the findDisease() method in the website’s JavaScript compares the generated binary number with 38 unique binary numbers to perform the final eye diagnosis.


Improvements to be made:
 Currently, the website diagnoses diseases based solely on words and not also on context. For instance, the phrases “I am lazy‘’ and “I have a lazy eye” gives the same result: Amblyopia (lazy eye). Though the sentences have different meanings, my program cannot understand the difference between them. I hope that I will be able to make my website better in the upcoming future.
